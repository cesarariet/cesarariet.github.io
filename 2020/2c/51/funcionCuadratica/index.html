Función Cuadrática

  • Forma Polinómica: \[ f(x) = ax^2+bx+c\]
  • Forma Canónica: \[f(x)= a(x-x_v)^2+y_v\]
  • Forma Factorizada: \[ f(x)= a(x-x_1)(x-x_2) \]

Ejemplos de funciones cuadráticas.

  • $f(x)=3x^2-2x+1$
  • $f(x) = x^2$
  • $f(x)= -(x+2)(x-4)$
  • $f(x)= x^2+2x(x+1)$

Resumen

  • $a$ es el coeficiente principal.
  • El gráfico de una función cuadrática es una parábola.
    • con ramas hacia arriba si $a\gt 0$
    • con ramas hacia abajo si $a\lt 0$.
  • $(x_v,y_v)$ es el vertice de la parábola. Desde la forma canónica se obtiene de: \[x_v=-\frac{b}{2a}, y_v=f(x_v)\]
  • $x_1$ y $x_2$ son las raices de la función cuadrática (si existen). Desde la forma canónica se obtienen de: \[ x_1,x_2=\frac{-b\pm\sqrt{b^2-4ac}}{2a} \]
  • De las raíces $x_1,x_2$ puede hallarse el vértice con: \[ x_v=\frac{x_1+x_2}{2}, y_v=f(x_v)\]

Forma polinómica

\[ f(x) = ax^2+bx+c \]

Ejemplos

  • $f(x)=3x^2+2x-1$ con $a=3,b=2,c=1$
  • $f(x)=-x^2+9$ con $a=-1,b=0,c=9$
  • $f(x)=x^2$ con $a=1,b=0,x=0$

Vértice de la parábola:\[ x_v=-\frac{b}{2a}, y_v=f(x_v)\]

Raíces o ceros de la cuadrática (si existen) \[ x_1,x_2=\frac{-b\pm\sqrt{b^2-4ac}}{2a} \]

Forma canónica

\[f(x)= a(x-x_v)^2+y_v\]

Ejemplos

  • $f(x)=3(x-4)^2-5$ con $a=3,x_v=4,y_v=-5$
  • $f(x)=(x+2)^2+3$ con $a=1, x_v=-2, y_v=3$
  • $f(x)= -x^2+9$ con $a=-1, x_v=0, y_v=9$
  • $f(x)= x^2$ con $a=1,x_v=0,y_v=0$
  • Para graficar una cuadrática es útil conocer el vértice.
  • Para pasar de forma polinómica $f(x)=ax^2+bx+c$ a la forma canónica usar: \[ x_v=-\frac{b}{2a}, y_v=f(x_v)\]
  • Para pasar de forma factorizada $f(x)=a(x-x_1)(x-x_2)$ a forma canónica usar \[ x_v=\frac{x_1+x_2}{2}, y_v=f(x_v)\]

Forma Factorizada

\[ f(x)=a(x-x_1)(x-x_2) \]

Ejemplos

  • $f(x)=3(x+2)(x-1)$ con $a=3,x_1=-2,x_2=1$
  • $f(x)=-(x+2)(x-3)$ con $a=-1, x_1=-2,x_2=3$
  • $f(x)=(x+3)x$ con $a=1,x_1=-3,x_2=0$
  • $f(x)=2(x-1)^2$ con $a=2,x_1=1,x_2=1$
  • $f(x)=x^2$ con $a=1,x_1=0,x_2=0$
  • La forma factorizada depende de que existan las raíces.
  • Para pasar de forma polinómica $f(x)= ax^2+b+x+c$ a factorizada \[ x_1,x_2=\frac{-b\pm\sqrt{b^2-4ac}}{2a} \]
  • Conocidas las raíces se puede hallar el vertice usando \[ x_v=\frac{x_1+x_2}{2}, y_v=f(x_v)\]
