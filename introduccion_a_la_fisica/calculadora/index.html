El separadador decimal

El separador decimal es un signo que se utiliza para indicar la separación de la parte entera y la parte decimal de un número.

El número 3,14 conocido en matemática como π tiene parte entera 3 y parte decimal 14, con el separador decimal coma (en este caso). Sin embargo, en otras culturas el separador decimal es el punto quedando 3.14 el número π.

Dependiendo el origen y como este configurada una calculadora tendrá la coma, el punto u otro símbolo como separador decimal.

Separadores de millares

Es común agrupar las cifras enteras de un número de tres en tres para falicitar su lectura y cada grupo separarlos por un punto (en argentina), una coma o un espacio.

Por ejemplo el numero 123456789 puede escribirse de las siguientes formas

  • 123.456.789 (separando miles y millares con un punto, muy usando en argentina)
  • 123,456,789 (separando miles y millares con una coma, usado en otros paises)
  • 123 456 789 (separando miles y millares con un espacio, de uso en argentina y otros paises)
  • 123456789 (Sin separadores)

Los separadores de millares tienen la función de facilitar la lectura pero debe evitarse el uso porque suelen confundirse con el separador decimal.

Las calculadoras y los separadores

Las calculadoras pueden estar configuradas o configurarse para mostrar separador decimal y separadores de millares. Cuando ambos separadores se encuentran disponibles suelen confundirse. Para reconocer que separadores tiene una calculadora debe tenerse en cuenta lo siguiente:

  1. El primer separador desde la derecha siempre es el separador decimal.
  2. En caso de mostrar separadores de millares, estos aparecen cada tres dígitos desde el separador decimal.
