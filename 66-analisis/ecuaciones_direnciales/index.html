Ecuaciones diferenciales

  • Ecuación que relaciona funciones y sus derivadas.
  • Cuando hay una sóla función desconcida (variable dependiente) y una sóla variable independiente, llamaremos a la expresión Ecuación diferencial ordinaria
  • Hay variados métodos de resolución para las ecuaciones diferenciales.

Método de separación de variables

Es un método para hallar las soluciones de una ecuación diferencial.

Consiste en separar ("despejar") las variables de una ecuación de forma que los miembros resultantes puedan integrarse.

Nota

  • En el miembro donde queda la función (digamos $f(x)$) intentar una sustitución igual a dicha función \[u=f'(x)\text{, } du=f'(x)dx\]
  • Si el problema tiene condición incial o de contorno, usarla para obtener la constante de integración.
  • Luego de integrar debe despejarse la función.
